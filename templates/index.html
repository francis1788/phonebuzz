PhoneBuzz

Welcome to PhoneBuzz! PhoneBuzz is a webapplication that uses the Twilio API to call a user, receive a numerical input from the user, and then recite the FizzBuzz sequence up to that input unto the user. To use this application one can call into the application, use the application to create an outgoing call, or replay a previous outgoing call in the call history.
Disclaimer: The Twilio account associated with this website is a free one, so the "Call from PhoneBuzz" option will not work.
